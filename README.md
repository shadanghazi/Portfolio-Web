# AI-Generated Portfolio Website

A modern, AI-tech themed portfolio site built entirely from an iteratively refined prompt. This repo documents the full prompt engineering process — from a basic first draft to a near-production-ready spec — along with the evaluation score at each stage.

**🔗 Live site:** [sheddix.netlify.app](https://sheddix.netlify.app/)

---

## 📈 Prompt Evolution

Each prompt version was scored out of 10 based on content completeness, clarity of goal, and overall effectiveness.

| Version | Score | Prompt File | Key Additions |
|:-------:|:-----:|:------------|:---------------|
| v1 | 6.5 / 10 | [`prompt-v1.md`](./prompt-v1.md) | Initial draft — name, skills, projects, contact, and profile links |
| v2 | 8.2 / 10 | [`prompt-v2.md`](./prompt-v2.md) | Added target audience, color palette direction, and a reference site |
| v3 | 9.0 / 10 | [`prompt-v3.md`](./prompt-v3.md) | Added a sections list, personal mission statement, a third skill, and fixed typos |
| v4 | 9.5 / 10 | [`prompt-v4.md`](./prompt-v4.md) | Added a landing hero section description and a "what to avoid" list |

**Score progression:** `6.5 → 8.2 → 9.0 → 9.5`

---

## 🔍 What Changed at Each Step

### v1 → v2 (6.5 → 8.2)
Added the three biggest missing pieces: **who my audience is**, a **color direction**, and a **reference site**. This single change accounted for the largest jump in score.

### v2 → v3 (8.2 → 9.0)
Added structure: a **sections list**, a **personal mission statement**, a **third skill**, and fixed lingering typos. Made the prompt cleaner and more specific.

### v3 → v4 (9.0 → 9.5)
Locked in the final details: a **landing hero section description** (matching the reference site's swipe/info layout) and a **"what to avoid"** list to prevent clashing colors and broken/empty pages.
